# jobtracker-api
A small backend portfolio project built to demonstrate practical experience with **C#/.NET**, **REST APIs** and a **relational database**.

## Tech stack
- C#/.NET (ASP.NET Core Web API)
- Entity Framework Core (code-first + migrations)
- SQL Server (local) / Azure SQL (cloud)
- Swagger/OpenAPI

## Goals
- Build a REST API with CRUD endpoints and proper HTTP status codes
- Persist data in a relational database using EF Core migrations
- Provide a clean local developer setup

## Definition of Done (v1)
- REST API (CRUD) + Swagger UI
- EF Core + migrations + SQL Server database
- Clear instructions to run locally
